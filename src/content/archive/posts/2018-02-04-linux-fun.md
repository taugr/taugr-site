The `cmatrix` and `sl` programs will add a bit of fun to your Linux shell.

## cmatrix

The Matrix has you:

![cmatrix](/img/2018-02-04/cmatrix.gif)

### Installation:

```bash
sudo apt install cmatrix
```

### Running

To start cmatrix just run:

```bash
cmatrix
```

Type `q` or `Ctrl+C` to exit.

Adjustments can be made by passing additional parameters. For a complete list run:

```bash
cmatrix -h
```

For example:

```bash
cmatrix -C yellow
```

turns the output yellow.

## sl

For those times when you mistype `ls` instead of seeing `sl: command not found` or similar why not display something a little more extravagant:

![sl](/img/2018-02-04/sl.gif)

`sl` instead standing for _steam locomotive_.

### Installation

```bash
sudo apt install sl
```

### Running

To run just mistype `ls`

```bash
sl
```

Some custom options:

```bash
sl -F   # the steam train flies
sl -l   # show a little one
```
