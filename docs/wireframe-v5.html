Why Agent Memory Architectures Are Still Wrong

Most agentic systems treat memory as a retrieval problem. You store everything, embed it, and hope the right context surfaces when you need it.

This is backwards.

The real problem is forgetting. Human memory works because it discards. The things that stick are the things that mattered. Everything else fades, and that's a feature, not a bug.

class MemoryStore: def remember(self, fact, importance): # Priority queue, not append-only log self.decay_old_memories() self.store(fact, weight=importance) def recall(self, query, k=5): # Recency + importance + relevance return self.weighted_search(query, k)
